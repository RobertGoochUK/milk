MILK FHIR Dosage-to-Text Dose-to-Product Translation Terminology Server

Project MILK (Medicines Interoperability and Logic Toolkit)

What tickles your fancy today..? Read the Release Notes.

FHIR Dosage-to-Text

Demonstrator web services to support the implementation of the FHIR STU3 Dosage structure within CareConnect profiled resources, to convert a structured dosage instruction into a human readable string.

Dose-to-Product Translation

Demonstrator web services using the NHS Dictionary of Medicines and Devices (dm+d) terminology to translation a dose-based instruction into a list of suitable products.

dm+d Terminology Server

Demonstrator web services for integration with the NHS Dictionary of Medicines and Devices (dm+d) terminology to return dm+d information.
